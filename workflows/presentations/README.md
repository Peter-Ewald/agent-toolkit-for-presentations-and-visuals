# Presentation Workflows

Use this folder for presentation-only tasks.

Current recipe types:

- Create a new Ramboll deck at a target path
- Apply the default Ramboll theme to an existing deck
- Refresh deck styling after changing the shared Marp theme

Implementation surface:

- `../../presentations/marp/`

Current command-level entry points:

- `../../presentations/marp/tools/create_default_ramboll_slide_deck.py`
- `../../presentations/marp/tools/apply_default_ramboll_theme_to_slide_deck.py`
