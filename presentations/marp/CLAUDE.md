# Marp Claude Notes

Use this folder for deck templates, Marp theme changes, and deck bootstrap/sync.

Read:

- `README.md`
- `../../docs/presentation-workflow.md`

Rules:

- Edit `themes/ramboll.css` for shared styling.
- Edit `templates/ramboll_default.md` for the shared default deck body.
- Use `tools/create_default_ramboll_slide_deck.py` to create new decks.
- Use `tools/apply_default_ramboll_theme_to_slide_deck.py` after shared style
	changes.
